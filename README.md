# system-software

## Overview
This repository contains assignments and practice work from the System Software course taken in the 2025 Spring semester. The focus is on low-level program behavior, assembly analysis, and system-level reasoning.

## Datalab
The purpose of this assignment is to become more familiar with bit-level representations of common patterns, integers, and floating-point numbers.

## Attacklab
Five buffer overflow attacks on two programs having different security vulnerabilities.
You will learn different ways that attackers can exploit security vulnerabilities.
For a better understanding of how to write programs that are more secure
- To understand the stack and parameter-passing mechanisms
- For a deeper understanding of how x86-64 instructions are encoded
